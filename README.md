# diamonddance.es

## Jekyll

### Install

Jekyll install from root directory

```bash
gem install bundler
bundle install
```

### Server

Jekyll server start

```bash
# local (development)
bundle exec jekyll serve --watch --config _config.yml

# production
JEKYLL_ENV=production bundle exec jekyll serve --watch
```
